# DevMystical Deobfuscator

Деобфускатор для файлов, написанных DevMystical.

## Возможности

- извлекает pickle-объекты из обфусцированного файла;
- восстанавливает AST;
- сохраняет восстановленный Python-код

## Использование

Запуск:

```bash
python deobf.py
```

После запуска укажи путь к файлу, который надо деобфусцировать.

Также можно использовать из Python:

```python
from deobf import deobfuscate_file

deobfuscate_file("obfuscated.py")
```

По умолчанию результат сохраняется рядом с исходным файлом с суффиксом `_deobf.py`.
